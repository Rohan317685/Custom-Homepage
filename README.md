# Custom-Homepage

This is a Custom Homepage for #resolution-coding! It has an integrated search bar and live analytics for weather and time along with a background that uses a [NASA API key](https://api.nasa.gov/#gibs) to change every single day. I made it using simple JS functions.

You can [try the live demo here](https://custom-homepage-blond.vercel.app/).

### Features:
* **Live Time Dashboard:** Updates every second using JS math.
* **Weather:** Real time stats! (might be wonky cause of some deployment issues).
* **Search Bar:** Quick access Google integration (built in search bar)
* **Daily NASA Background:** New space imagery every 24 hours.Updated using nasa api key

**The goal of the project:** To implement and learn about how API keys work and how to handle async data!
